B2C Accelerator with OMS 
For Kyma integration + ApiRegistry, optionally, ccv2.services.api.url.0 property should be set to a server url reachable by kyma instead of https://localhost:9002.
 Platform Setup:
   1. Setup platform using command ./install.sh -r b2c_b2b_acc_oms setup
   2. Initialize platform using command ./install.sh -r b2c_b2b_acc_oms initialize
   3. Start platform using command ./install.sh -r b2c_b2b_acc_oms start
